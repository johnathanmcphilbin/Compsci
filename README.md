# Compsci

## Run locally

Preview the site locally with a simple static server.

Using Python 3 (no install required):

```bash
python3 -m http.server 8000
```

Open your browser to: http://localhost:8000

Or, using Node (if you have Node.js installed):

```bash
npx http-server -p 8000
```

This will serve the project folder so you can view `index.html` and verify styling and scripts.
