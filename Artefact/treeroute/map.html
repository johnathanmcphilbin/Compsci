Model — why this meets requirements
- Uses embedded micro:bit sensor data (Moisture) plus sample and county data.
- Python model computes a wildfire-risk index (0–1) and runs two what-if scenarios (+3°C, -30% rain).
- Scenario deltas show how risk changes per point.
